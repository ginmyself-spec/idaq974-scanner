AdvantechAE iDAQ-974 ip scanning tool ---
---------------------
---------------------
Requre pakcage:
1. Python
2. python "tkinter" package
3. python "customtkinter" package

---------------------
---------------------
SOP:
1. Install Advantech DAQNavi Linux driver and SDK first.
2. Confirm if requirement package ready.
3. run this tool.

Run tool in virtual environment
1. Build up the environment
>>python -m venv venv
2. Run the venv
	- In Windows system
	>>venv\Scripts\activate
	-In Linux system
	>>source venv/bin/activate
3. Install necessary package
>>pip install customtkinter 
4. Execute tool
>>python AdvantechAE_iDAQ974_Scanning_v2
